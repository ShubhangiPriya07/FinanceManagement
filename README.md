## Project Description
Financify is a simple and interactive personal budgeting web application built using HTML, CSS, and JavaScript. It helps users understand, track, and control their financial 
activities by recording income and expenses, categorizing spending, and displaying real-time financial summaries. The project focuses on DOM manipulation and client-side data 
handling using localStorage.

---

## Problem Statement
Managing personal finances is often difficult due to lack of awareness about income, expenses, and spending patterns. Many individuals do not clearly understand where their 
money goes, leading to poor financial decisions. This project aims to solve that problem by providing a simple budgeting tool that visually represents income, expenses, and 
balance, helping users make informed financial decisions.

---

## Features Implemented
- Add income with source and amount
- Add expenses with name, amount, and category
- Automatic calculation of total income, total expenses, and balance
- Category-wise expense analysis
- Transaction history with delete functionality
- Reset button to clear all data
- Persistent data storage using localStorage
- Visual indicators for income (green), expenses (red), and balance status

---

## DOM Concepts Used
- `getElementById` and `querySelector` for element selection
- Event handling using `addEventListener`
- Dynamic element creation using `createElement`
- Updating content using `innerHTML` and `textContent`
- Conditional styling using `classList` and inline styles
- DOM updates based on user input
- Event-driven UI rendering
- Basic asynchronous function usage for UI rendering

---

## Known Limitations
- No user authentication or multiple user support
- No data visualization charts (only text-based analysis)
- Data is stored locally and not synced across devices
- No monthly or yearly budgeting separation
- Not optimized for very small screen sizes
